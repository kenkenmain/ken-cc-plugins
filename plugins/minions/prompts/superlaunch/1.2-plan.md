# Phase 1.2: Plan [PHASE 1.2]

## Subagent Config

- **Primary:** subagents:planner (parallel batch, 1-10 agents)
- **Supplementary:** `subagents:architecture-analyst` (1 agent, parallel with primary)
- **Aggregator:** `subagents:plan-aggregator`
- **Output:** `.agents/tmp/phases/1.2-plan.md` (written by aggregator, NOT by orchestrator)

## Dispatch Instructions

### Step 1: Dispatch Primary + Supplementary Agents

1. Read brainstorm results to identify implementation areas
2. For each planner agent i (1-indexed, matching area number), include in its dispatch prompt:
   `Temp output file: .agents/tmp/phases/1.2-plan.planner.{i}.tmp`
3. For architecture-analyst, include in its dispatch prompt:
   `Temp output file: .agents/tmp/phases/1.2-plan.architecture-analyst.tmp`
4. Dispatch one planner agent per area **and** `subagents:architecture-analyst` in a single parallel Task tool message

### Step 2: Dispatch Aggregator

5. **After ALL Step 1 agents complete**, dispatch `subagents:plan-aggregator`
6. The aggregator reads all `1.2-plan.*.tmp` files, renumbers tasks, resolves dependencies, and writes `1.2-plan.md`

**Do NOT read any agent results.** The aggregator handles all merging, renumbering, and writing.
