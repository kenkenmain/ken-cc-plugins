# Phase 2.3: Implementation Review [PHASE 2.3]

## Subagent Config

- **Type:** codex-reviewer (uses codex-high MCP)
- **Input:** `.agents/tmp/phases/1.2-plan.md`, git diff
- **Output:** `.agents/tmp/phases/2.3-impl-review.json`

## Instructions

Review implementation against the plan using Codex MCP.

### Process

1. Get list of modified files from `.agents/tmp/phases/2.1-tasks.json`
2. Dispatch to Codex MCP (codex-high) with implementation review criteria from `prompts/high-stakes/implementation.md`
3. Write review result to output file

### Codex Dispatch

```
mcp__codex-high__codex(
  prompt: "Review implementation. Plan: .agents/tmp/phases/1.2-plan.md. Modified files: [list]. Criteria: prompts/high-stakes/implementation.md. Return JSON with status, issues[], filesReviewed, summary.",
  cwd: "{project dir}"
)
```

### Output

Write Codex response as JSON to `.agents/tmp/phases/2.3-impl-review.json`

### If Issues Found

Same retry logic as Phase 1.3: fix, re-review, repeat until approved or max retries.
