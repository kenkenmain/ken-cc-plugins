# Phase 4.3: Completion [PHASE 4.3]

- **Type:** Bash
- **Input:** `.agents/tmp/phases/4.2-final-review.json`
- **Output:** `.agents/tmp/phases/4.3-completion.json`

## Instructions

Complete the workflow: create git branch and PR if configured.

### Process

1. Read final review result
2. If `readyForCommit: true`:
   - Create feature branch (if not already on one)
   - Stage changed files (exclude `.agents/**`)
   - Create commit with summary
   - Push and create PR (if gitWorkflow: "branch+PR")
3. Write completion result to output file

### Output Format

Write to `.agents/tmp/phases/4.3-completion.json`:

```json
{
  "status": "completed",
  "branch": "feat/task-name",
  "commit": "abc1234",
  "pr": { "number": 42, "url": "..." }
}
```
