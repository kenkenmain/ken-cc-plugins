# Phase 1.2: Plan [PHASE 1.2]

## Subagent Config

- **Type:** Plan (parallel batch, 1-10 agents)
- **Input:** `.agents/tmp/phases/1.1-brainstorm.md`
- **Output:** `.agents/tmp/phases/1.2-plan.md`

## Instructions

Dispatch parallel Plan agents to create a detailed implementation plan.

### Process

1. Read `.agents/tmp/phases/1.1-brainstorm.md`
2. Identify distinct planning areas (database, API, frontend, tests, etc.)
3. Dispatch one Plan agent per area
4. Merge results into structured plan

### Plan Schema (task-dispatcher readable)

Each task MUST follow this format:

```
### Task N: {name}

- **Files:** comma-separated paths
- **Dependencies:** none | Task M, Task N
- **Complexity:** easy|medium|hard
- **Instructions:** multi-line implementation guidance
```

### Output Format

Write to `.agents/tmp/phases/1.2-plan.md`:

```
# Implementation Plan

## Overview
{summary}

## Tasks

### Task 1: ...
...

## Dependency Graph
{task ordering}
```

### Validation

- All tasks have required fields
- Dependencies reference valid task IDs
- No circular dependencies
