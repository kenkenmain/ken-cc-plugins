# Phase 2.2: Simplify [PHASE 2.2]

## Subagent Config

- **Type:** code-simplifier
- **Input:** `.agents/tmp/phases/2.1-tasks.json`
- **Output:** `.agents/tmp/phases/2.2-simplify.md`

## Instructions

Review implemented code for simplification opportunities.

### Process

1. Read `.agents/tmp/phases/2.1-tasks.json` to get list of modified files
2. Review each modified file for:
   - Unnecessary complexity
   - Duplicate code
   - Over-engineering
   - Missing simplifications
3. Apply simplifications directly
4. Write summary to output file

### Output Format

Write to `.agents/tmp/phases/2.2-simplify.md`:

```
# Simplification Report

## Changes Made
- {file}: {description of simplification}

## No Changes Needed
- {file}: already clean
```
