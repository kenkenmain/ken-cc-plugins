# Phase 1.3: Plan Review [PHASE 1.3]

## Subagent Config

- **Type:** `state.reviewer` (codex-reviewer or claude-reviewer)
- **Criteria:** `prompts/high-stakes/plan-review.md`
- **Input:** `.agents/tmp/phases/1.2-plan.md`
- **Output:** `.agents/tmp/phases/1.3-plan-review.json`

## Instructions

Review the implementation plan against high-stakes criteria.

### Process

1. Read the review criteria from `prompts/high-stakes/plan-review.md`
2. Read the plan from `.agents/tmp/phases/1.2-plan.md`
3. Evaluate the plan against every criterion in the checklist
4. Write structured JSON result to `.agents/tmp/phases/1.3-plan-review.json`

### Output

Write JSON to `.agents/tmp/phases/1.3-plan-review.json` with: `status`, `issues[]`, `summary`

### If Issues Found

If status is "needs_revision" with blocking issues:

1. Fix issues based on suggestions
2. Re-run review
3. Repeat until approved or max retries (config: retries.maxPerPhase)
