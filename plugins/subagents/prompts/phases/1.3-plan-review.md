# Phase 1.3: Plan Review [PHASE 1.3]

## Subagent Config

- **Type:** codex-reviewer (uses codex-high MCP)
- **Input:** `.agents/tmp/phases/1.2-plan.md`
- **Output:** `.agents/tmp/phases/1.3-plan-review.json`

## Instructions

Review the implementation plan using Codex MCP.

### Process

1. Read `.agents/tmp/phases/1.2-plan.md`
2. Dispatch to Codex MCP (codex-high) with plan-review criteria from `prompts/high-stakes/plan-review.md`
3. Write review result to output file

### Codex Dispatch

```
mcp__codex-high__codex(
  prompt: "Review the implementation plan at .agents/tmp/phases/1.2-plan.md using criteria from prompts/high-stakes/plan-review.md. Return JSON with status, issues[], summary.",
  cwd: "{project dir}"
)
```

### Output

Write Codex response as JSON to `.agents/tmp/phases/1.3-plan-review.json`

### If Issues Found

If status is "needs_revision" with blocking issues:

1. Fix issues based on suggestions
2. Re-run review
3. Repeat until approved or max retries (config: retries.maxPerPhase)
