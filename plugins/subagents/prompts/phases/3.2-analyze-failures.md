# Phase 3.2: Analyze Failures [PHASE 3.2]

## Subagent Config

- **Type:** general-purpose
- **Input:** `.agents/tmp/phases/3.1-test-results.json`
- **Output:** `.agents/tmp/phases/3.2-analysis.md`

## Instructions

Analyze test failures and suggest fixes.

### Process

1. Read `.agents/tmp/phases/3.1-test-results.json`
2. If all tests passed: write "All tests passed" and exit
3. If failures: analyze error output, identify root causes, suggest fixes
4. Optionally apply fixes directly

### Output Format

Write to `.agents/tmp/phases/3.2-analysis.md`:

```
# Test Analysis

## Status
{passed|failed}

## Failures

### Failure 1: {test name}
- Error: {message}
- Root cause: {analysis}
- Fix: {suggestion or "applied"}

## Summary
{overall assessment}
```
