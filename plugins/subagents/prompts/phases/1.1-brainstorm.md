# Phase 1.1: Brainstorm [PHASE 1.1]

## Subagent Config

- **Type:** general-purpose
- **Input:** `.agents/tmp/phases/0-explore.md`
- **Output:** `.agents/tmp/phases/1.1-brainstorm.md`

## Instructions

Read the explore findings and determine the implementation approach.

### Process

1. Read `.agents/tmp/phases/0-explore.md`
2. Analyze patterns, existing code, and constraints
3. Determine 2-3 approaches with trade-offs
4. Select recommended approach
5. Write decisions to output file

### Output Format

Write to `.agents/tmp/phases/1.1-brainstorm.md`:

```
# Brainstorm Results

## Task
{task description}

## Explore Findings Summary
{key findings}

## Approaches Considered

### Approach 1: {name}
- Pros: ...
- Cons: ...

### Approach 2: {name}
...

## Selected Approach
{chosen approach with rationale}

## Implementation Areas
{list of distinct areas to plan}
```
