# Phase 3.1: Run Tests [PHASE 3.1]

## Subagent Config

- **Type:** Bash
- **Input:** test commands from config
- **Output:** `.agents/tmp/phases/3.1-test-results.json`

## Instructions

Run lint and test commands, capture results.

### Process

1. Run lint command (default: `make lint`)
2. Run test command (default: `make test`)
3. Capture exit codes, stdout, stderr
4. Write results to output file

### Output Format

Write to `.agents/tmp/phases/3.1-test-results.json`:

```json
{
  "lint": { "exitCode": 0, "stdout": "...", "stderr": "" },
  "test": { "exitCode": 0, "stdout": "...", "stderr": "" },
  "allPassed": true
}
```
